## 🤖 Platform at a glance

| Parameter | Value |
|-----------|-------|
| Drive type | Differential drive |
| Chassis | Acrylic plate |
| Onboard computer | Raspberry Pi 5 |
| Motor driver | L298N dual H-bridge |
| Power | 11.1 V Li-ion battery |
| Optical flow sensors | 2 × USB mouse, 8000 DPI |
| OFS baseline | d = 99.51 mm |
| IMU | SmartElex ISM330DHCX |
| Range sensors | 4 × HC-SR04 ultrasonic |
| Fusion rate | 100 Hz |
